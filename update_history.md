# 更新履歴 (Update History)

## [2026-03-29] ASSTステータス算出ロジックの刷新とUI表示の変更

- **変更理由**: 防具検索結果のステータス表示を武器の基礎値に依存せず、0ベース（武器・スキル・防具の合計）で計算するようにし、期待値計算を攻撃力と会心率のみに基づくシンプルな方式に変更するため。
- **実施内容**:
  - `js/asst.js` の `calculateFinalStats` 関数をリファクタリング。ハードコードされた初期値（330等）を廃止し、URLパラメータ `base_atk`, `base_aff` または 0 から計算を開始するように変更。
  - 期待値の計算式を `攻撃力 * (1 + 0.25 * 会心率 / 100)` に簡略化。
  - `js/asst.js` の `renderResult` 内の `result-header` を更新し、ユーザー指定の1行表示フォーマット（期待値、攻撃力、会心率、防御力、耐性）に変更。
  - `js/main.js` の検索ボタン実行時に、現在の計算機の武器基礎攻撃力と会心率を ASST ページに引き継ぐようパラメータを追加。
- **影響範囲**: 防具検索（ASST）の結果表示、およびメインページからの検索遷移。


## [2026-03-28] 開発環境の整理整頓とフォルダ構成の刷新

プロジェクトのルートディレクトリをクリーンに保つため、以下のルールに基づきファイルを移動・整理しました。
